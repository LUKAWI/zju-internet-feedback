# Danmaku 弹幕系统 - 验证清单

## 功能验证

- [ ] 弹幕组件成功加载并显示 3-5 条弹幕
- [ ] 弹幕从右侧进入，左侧退出，持续滚动
- [ ] 多条弹幕同时滚动，高度随机分布
- [ ] 弹幕内容超过 30 字符时截取显示
- [ ] hover 时弹幕暂停并显示完整内容提示
- [ ] 刷新页面展示不同的留言组合
- [ ] 弹幕循环播放，内容持续变化

## 设计验证（VS Code Light 配色）

- [ ] 弹幕背景色为 #f3f3f3（VS Code Light）
- [ ] 弹幕文字色为 #3c3c3c（VS Code 文字色）
- [ ] 弹幕边框色为 #d4d4d4（VS Code 边框色）

## 设计验证（社交媒体文本框风格）

- [ ] 弹幕采用圆角卡片设计（圆角 8px）
- [ ] 弹幕具有轻微阴影效果
- [ ] 弹幕布局为：头像 + 用户名 + 内容
- [ ] 用户名显示在内容上方或旁边
- [ ] 头像为圆形 24x24 像素

## 响应式验证

- [ ] 桌面端弹幕容器高度适配
- [ ] 移动端弹幕容器高度适配
- [ ] 字体大小自适应
- [ ] 无横向溢出

## 性能验证

- [ ] FPS 保持 60 帧
- [ ] 无明显卡顿
- [ ] 内存使用稳定

## 代码质量验证

- [ ] Danmaku 组件独立封装
- [ ] useDanmaku hook 逻辑清晰
- [ ] 使用 IntersectionObserver 优化
- [ ] 无内联组件定义
- [ ] 符合 React Best Practices

## 页面集成验证

- [ ] HeroTitle 组件已移除
- [ ] 副标题已移除
- [ ] Danmaku 组件正确集成
- [ ] 搜索栏和排序导航位置正确
