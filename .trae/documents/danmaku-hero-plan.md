# Hero 区域弹幕效果修改计划

## 需求分析

用户希望：
1. **移除**当前的大标题「校园网反馈」和副标题「ZJUers，分享你的校园网体验」
2. **添加**弹幕效果：从用户留言中选取一部分作为弹幕，从右至左飘过该区域

## 修改方案

### 1. 创建弹幕组件 `components/Danmaku.tsx`
- 从 `/api/messages` 获取留言数据
- 选取部分留言内容作为弹幕（截取前 25-30 字符）
- 弹幕从右侧进入，左侧退出，持续滚动
- 多条弹幕同时滚动，高度随机分布
- hover 时弹幕暂停，显示完整内容提示

### 2. 修改 `app/page.tsx`
- 移除 HeroTitle 组件
- 移除副标题段落
- 添加 Danmaku 组件
- 搜索栏和排序导航上移

### 3. 添加弹幕动画 CSS（globals.css）
- `@keyframes danmaku-scroll`：从右向左滚动动画
- `.danmaku-container`：弹幕容器，overflow-hidden
- `.danmaku-item`：弹幕项，absolute定位

## 文件修改清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `components/Danmaku.tsx` | 新建 | 弹幕组件 |
| `app/page.tsx` | 修改 | 移除标题，添加弹幕 |
| `app/globals.css` | 修改 | 添加弹幕动画样式 |

## 弹幕设计

- **容器高度**：120px（桌面端）/ 80px（移动端）
- **弹幕数量**：同时显示 3-5 条
- **滚动速度**：8-12 秒/条
- **内容截取**：前 25 字符，hover 显示完整内容
- **样式**：纯黑白，无边框，极简设计

## 实现步骤

1. 添加弹幕动画 CSS 到 globals.css
2. 创建 Danmaku 组件
3. 修改 page.tsx 集成弹幕
4. 测试效果
